# Multi-Channel Marketing Analytics Project

## Overview
This project analyzes advertising performance across Facebook, Google, and TikTok by unifying campaign data into a single dataset and building an interactive dashboard for cross-channel performance analysis.

## Objective
The goal of this project is to provide a clear and actionable view of marketing performance, helping stakeholders identify opportunities to optimize budget allocation and improve conversion efficiency.

## Data Sources
- Facebook Ads data
- Google Ads data
- TikTok Ads data

## Approach

### 1. Data Preparation
- Standardized schema across all platforms
- Cleaned and aligned key fields such as impressions, clicks, cost, and conversions
- Unified multiple datasets into a single analytical structure

### 2. SQL Transformation
- Combined platform-level datasets into one table
- Ensured consistent metric definitions across sources
- Prepared data for reporting and visualization

### 3. KPI Design
The following key metrics were used:
- CTR (Click Through Rate)
- CPC (Cost Per Click)
- CPM (Cost Per Thousand Impressions)
- CPA (Cost Per Acquisition)
- ROAS (Return on Ad Spend)
- Conversions
- Cost

### 4. Dashboard Development
Developed an interactive Power BI dashboard to:
- Compare performance across platforms
- Analyze campaign-level results
- Evaluate engagement and trends over time
- Enable filtering by campaign, ad group, and time

## Key Insights
- TikTok drives the highest number of conversions but also consumes the largest share of the budget
- Some campaigns consistently deliver strong performance and are good candidates for scaling
- High engagement does not always translate into higher conversions
- Budget efficiency can be improved by reallocating spend from high-cost, lower-return campaigns
